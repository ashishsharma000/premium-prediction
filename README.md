#  Insurance Premium Prediction System

##  Overview
This is an end-to-end Machine Learning project that predicts insurance premiums based on user inputs like age, BMI, smoking status, and region.

##  Features
- Machine Learning model using Scikit-learn
- Data preprocessing using ColumnTransformer & Pipeline
- FastAPI backend for prediction
- Streamlit frontend for user interaction
- Feature engineering (BMI from height & weight)

## 🛠 Tech Stack
- Python
- Scikit-learn
- FastAPI
- Streamlit
- Pandas, NumPy

##  How to Run

### 1. Clone the repository

### 2. Install dependencies

### 3. Run Streamlit app

### 4. Run FastAPI

##  Output
Predicts insurance premium based on user inputs.

##  Project Structure
- main.py → FastAPI backend  
- streamlit.py → Frontend UI  
- insurance_pipeline.pkl → Trained model  
- requirements.txt → Dependencies
